
## Nome do componente

    Caminho do componente: (exemplo: @/components/molecules/CardNavigationApp)

---

Ele foi criado para ser um componente dinâmico que é possível adicionar uma foto com tamanho dinâmico ou algum icone com cor de fundo dinâmicas.

---

| Nome da prop. | Tipo | Padrão  |  Descrição
|--|--|--|--|--|
| ```size``` | number | Tamanho do componente define width e height |
| ```title``` |  string | 80 | Se tiver um título, será acrescido na parte inferior do componente |
| ```iconName``` | string | Nome do icone |
| ```typeIcon``` | string | Fonte do Icone |
| ```route``` | string | Rota que será destinada caso for necessário |
| ```bgColor``` | string | Cor de fundo |
| ```source``` | string | Source da imagem caso for necessário |
| ```mgTop``` | number | 16 | Margin Top Caso for necessário |
